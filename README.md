# acc102-track2
This is a Student monthly consumption data analysis using Python, including data cleaning, statistics, and visualization,completed for ACC102 mini assignment.

## 1. Project Purpose
To analyze student spending patterns and understand the monthly consumption structure of university students.

## 2. Data Information
- Data type: Student monthly consumption survey data
- Fields: gender, grade, monthly_budget, food, shopping, entertainment, transportation, study, total_consumption
- Data size: 50 student records

## 3. Methods & Tools
- Python (Pandas) for data cleaning and analysis
- Matplotlib for data visualization
- Jupyter Notebook for coding

## 4. Key Findings
- Food is the largest part of student consumption.
- Female students spend slightly more than male students on average.
- Monthly budget is positively related to total consumption,Monthly living expenses are positively correlated with total consumption.

## 5. Files in this repo
- student_consumption_analysis.ipynb: main code notebook
- student_consumption_real.csv: cleaned data file
- analysis_plots.png: output charts

## 6. How to run
1. Open the .ipynb file with Jupyter Notebook
2. Run all cells，Click "Run" to view the analysis results and charts

## 7. Limitations
- Sample size is small (50 students)
- Data is simulated based on real campus scenarios
- Does not include regional differences
- The analysis is rather limited in scope.
